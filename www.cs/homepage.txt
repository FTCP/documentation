Dokonalé zabezpečení
------------
Nette používá revoluční technologii, která eliminuje výskyt bezpečnostních děr a jejich zneužití, jako je např. XSS((Cross-site scripting)), CSRF((Cross-site request forgery)), session hijacking, session fixation atd. [Chcete vědět více? | doc:vulnerability-protection]


[*  .[icon] *]

Ladicí nástroje
----------
Disponuje bezkonkurenčními [ladicími nástroji | doc:debugging], které vám pomohou zavčasu odhalit všechny chyby aplikace.


[*  .[icon] *]

Exceluje ve výkonu
---------------
Dle [nezávislého testu | http://www.root.cz/clanky/velky-test-php-frameworku-zend-nette-php-a-ror/] je Nette Framework jedním z nejrychlejších frameworků vůbec.


[*  .[icon] *]

Nejaktivnější komunita v ČR
---------
Nebudete v tom sám! [Komunita] kolem frameworku radí, vytváří doplňky a rozšíření, sdílí své zkušenosti.


[*  .[icon] *]

Strmá křivka učení
----------------------
Naučíte se tvořit webové aplikace za velmi krátký čas.


[*  .[icon] *]

Dobré návyky
------------
Nette Framework vás vede k čistému návrhu aplikace s důrazem na budoucí rozšiřitelnost.

Moderní framework
------
AJAX((Asynchronous JavaScript and XML)) / AJAJ((Asynchronous JavaScript and JSON)), Dependency Injection, SEO((Search engine optimization)), DRY((Don't repeat yourself)), KISS((Keep it short and simple)), MVC((Model-View-Controller)), Web 2.0, cool URL - propracovaná podpora všech moderních technologií a koncepcí.


[*  .[icon] *]

Vyzrálý objektový návrh
---------
Promyšlený a čistý objektový návrh využívající nových vlastností PHP 5, komponent a událostmi řízeného modelování.


[*  .[icon] *]

Pluginy a rozšíření
------------
Neustále se rozrůstající [nabídka doplňků | addons:homepage] pro snadné použití ve vašich aplikacích. Oceňte výhody znovupoužitelného kódu.


[*  .[icon] *]

Open-source licence
-----------
[Licence BSD | license] patří k těm nejvolnějším. Framework můžete zdarma používat i v komerčních projektech.


[*  .[icon] *]

Práce v týmu
---------
Výborně rozděluje práci mezi více programátorů a HTML kodérů.


[*  .[icon] *]

Dává vám volnost
--------------
