Development
***********

.[perex]
Nette Framework is an open source project to which anyone can contribute.


[*  .[icon] *]

"Git Repository":https://github.com/nette/nette
--------------

The latest development version of the framework can be obtained from the official Git repository available at https://github.com/nette/nette.

If you want to contribute, please respect the [Commit Policy].


[*  .[icon] *]

"Issue Tracker":https://github.com/nette/nette/issues
----------------

Issue tracker is used for reporting and overview of bugs. Each [repository | https://github.com/nette] has its own issue tracker. If you are not sure whether certain behavior is a bug, try discussing it on [forum | http://forum.nette.org/en/] first.
